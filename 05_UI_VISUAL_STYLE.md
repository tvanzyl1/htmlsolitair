# 05 — UI, Layout, and Visual Style Agent

## Mission

Translate the attached generic style template into a solitaire-appropriate visual identity. fileciteturn0file0

## Core visual direction

Keep the game:
- polished
- modern
- colourful
- readable
- premium-feeling
- softly playful

Adapt the style template into something more elegant than chaotic:
- rich dark table/background base
- vibrant accent colours
- rounded UI panels
- subtle gradients and glows
- clean shadow layering
- crisp card faces
- satisfying button and menu animation

## Do not make it feel like:
- a fake gambling app
- an old Windows 95 clone
- a bland spreadsheet of cards
- a noisy neon overload that hurts readability

## Layout requirements

The main game screen should clearly show:
- stock
- waste
- foundations
- tableau
- top HUD / controls

Suggested layout:
- top row: stock, waste, spacer, four foundations
- centre/lower area: seven tableau columns
- header or floating HUD for timer, moves, score, sound, restart, new game, undo placeholder

## Menus / overlays

Create polished overlays for:
- start screen
- pause/settings if included
- win screen
- confirm restart if needed

Style guidance:
- big clear title
- elegant card-table atmosphere
- strong CTA buttons
- flavour subtitle
- rounded panels and soft shadow depth
- mobile-friendly spacing

## Card design

Cards should be:
- highly legible
- easy to distinguish by suit and colour
- large enough on mobile
- visually layered with clean borders and shadows

Recommended card states:
- face-down
- face-up
- selected
- draggable
- target-highlight
- auto-moving
- invalid-shake

## Responsive design

### Desktop
- generous spacing
- easy drag-and-drop
- wide tableau columns

### Tablet
- slightly compressed spacing
- cards still comfortably readable
- controls remain obvious

### Phone
- prioritise readability over fitting too much empty space
- consider slight scaling and tighter pile spacing
- ensure players can still identify ranks and suits quickly

## UI copy tone

Use light, polished, cheerful text:
- “New Game”
- “You Won!”
- “Nice move”
- “No move there”
- “Try again”
- “Single Draw”
- “Tap stock for next card”

Avoid overly jokey copy that breaks the elegant solitaire mood.

## Deliverables

Produce:
- visual design brief
- layout structure
- responsive breakpoints
- component list
- card state style list
- menu / overlay style direction
