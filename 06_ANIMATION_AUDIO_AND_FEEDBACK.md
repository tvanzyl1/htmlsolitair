# 06 — Animation, Audio, and Feedback Agent

## Mission

Add juice and feedback consistent with the attached generic style template, but adapted to solitaire. fileciteturn0file0

This is not an action game, so feedback should be:
- elegant
- satisfying
- clear
- restrained but delightful

## Required feedback areas

### Card movement
- smooth lift on pick-up
- crisp snap into place on valid drop
- graceful return on invalid move
- mild emphasis when auto-moving to foundation

### Selection states
- clear glow, outline, or shadow state
- subtle pulse for selected cards
- destination highlights for legal targets

### Button interactions
- hover animation on desktop
- press animation on all platforms
- polished transitions between enabled/disabled states

### Stock draw
- satisfying card flip or slide to waste
- subtle audio tick / flick if enabled
- quick but readable animation

### Foundation progress
- pleasant reinforcement when a card reaches foundation
- small sparkle, glow pulse, or celebratory micro-effect
- stronger feedback for completing a suit

### Win state
- celebratory but tasteful
- optional card cascade, confetti, shimmer, or victory sweep
- overlay with replay CTA
- maybe a classic solitaire-inspired flourish in a modern toned-down way

## Sound guidance

Optional but recommended.
If sound is included:
- soft card slide
- light place / confirm sound
- subtle invalid sound
- tasteful win jingle

Avoid:
- loud casino sounds
- noisy arcade spam
- repetitive harsh effects

Include a mute toggle.

## Motion guidelines

- keep animations short and responsive
- prefer easing that feels soft and polished
- reduced motion should remain playable and pleasant
- never let animations slow down core interaction

## Micro-feedback ideas

- move counter increments with a tiny pulse
- timer updates cleanly
- undo placeholder can show disabled polish even if not implemented yet
- hint / safe auto button can pulse very gently when useful in later versions

## Deliverables

Produce:
- animation list
- timing recommendations
- sound list
- reduced-motion notes
- win celebration plan
- feedback mapping for every major player action
